# psp japanese helloworld

[詳細](https://511v41.github.io/2021/05/02/psp)

## How to build

私の書いた[記事](https://511v41.github.io/2021/04/30/psp)を参考に開発環境を構築してください。

あとは `make` するだけ。

出力される `EBOOT.PBP` が本体なので、これをエミュレータなり実機なりに突っ込めば動くはずです。
